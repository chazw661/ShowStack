CSV Layout

The first row must be a header. Column order does not matter and matching is case-insensitive. Rows are added to the currently-selected project. New zone labels are auto-created.

CableYesOne of the codes or display names listed below
LengthYesInteger feet per run
NotesNoFree text
Drawing RefNoFree text
Fan OutsNoTYPE xN, multiple separated by ;
ExtensionsNoFAN_TYPE: EXT_CABLE LENGTH xN, semicolon-separated
CouplersNoTYPE xN, semicolon-separated
