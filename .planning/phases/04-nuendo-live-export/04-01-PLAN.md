    ```python
    @property
    def resolved_yamaha_name(self):
        """D-04: Yamaha palette name for Nuendo Farb mapping, or None.

        Resolution order (per Phase 4 CONTEXT D-04 / D-05):
          1. color_override hex → reverse-lookup against YAMAHA_TO_HEX.
             If override is a non-palette hex (e.g. '#A1B2C3'), return
             None → exporter omits <int name='Farb'/> per D-05.
          2. resolved_source.color (channel-level YAMAHA_COLOR_CHOICES
             name on ConsoleInput / ConsoleAuxOutput / ConsoleMatrixOutput
             / ConsoleStereoOutput — all added in Phase 2 D-07).
          3. None.

        Returning None signals 'omit Farb' to the Nuendo exporter. Phase 1's
        resolved_color (hex) is intentionally NOT touched here — that
        contract underpins the byte-stable Reaper exporter.
        """
        if self.color_override:
            name = _HEX_TO_YAMAHA_NAME.get(self.color_override.lower())
            if name:
                return name
            return None  # D-05: hex override but not a palette match
        src = self.resolved_source
        if src is None:
            return None
        color = getattr(src, 'color', None)
        if not color or color == 'Off':
            return None
        return color
    ```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Engineer swatch input → `MultitrackTrack.color_override` | Hex string flows from the editor's color picker (already validated by Phase 1's AJAX endpoint) into the DB; this plan reads but does not write that field. |
| `_YAMAHA_TO_HEX` → `_HEX_TO_YAMAHA_NAME` import boundary | The reverse-lookup map is built once at module load from a trusted constant in `planner/utils/reaper_export.py`; no user data participates in the build. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-01 | Information Disclosure | `_HEX_TO_YAMAHA_NAME` | accept | Static palette map (8 well-known engineering colors); not sensitive. |
| T-04-02 | Tampering | `color_override` hex read inside `resolved_yamaha_name` | mitigate | `.lower()` normalization + dict `.get()` lookup; any non-matching string (including malicious payloads like `'); DROP TABLE—'`) returns `None` cleanly. No SQL, no eval, no f-string interpolation of the value. |
| T-04-03 | DoS | Module-load reverse-map build | accept | One dict comprehension over a 10-entry constant. Cost is negligible (<1ms) and runs once per Python process. |
</threat_model>
